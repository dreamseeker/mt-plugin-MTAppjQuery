MTAppjQuery - Movable Type Plugin
=================

[English](README.md)

## 概要

Movable Type の管理画面を jQuery の簡単な記述でカスタマイズすることができるプラグインです。Perl の知識は不要です。

## 動作条件

* Movable Type 6
